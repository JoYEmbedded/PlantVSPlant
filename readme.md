## 游戏启动界面

1. 播放背景音乐
2. 按下任意键切换到角色选择界面（不要按回车键，目前存在bug懒得修复，这会导致直接跳过用户选择界面）

1. 按下左右键能够选择角色，按下确认键后进入战斗界面
2. 动态背景及角色动画的实现。背景中的阴影始终呈现对手角色。
3. 切换角色播放切换的音乐
